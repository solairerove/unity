Add a worker profile

To add a worker profile click the button "create worker" on the worker tab. You are able to load pictures that have size less than 15mb. Some of the fields are required. Required fields are:
-First name
-Last name
-Gender
-Speciality
Other fields are optional, but it is highly recommended to fulfill "Description" and "Skype". First one is needed for faster access to worker professional level, second one is for communication (for example if he is a remote worker).

Delete a worker profile or profiles

You can delete like so single worker or group of workers. Mark the workers that are no needed more, and click "Delete workers" button. Then think once again and click "Delete" in new modal. Also you can delete single user manualy by clicking actions button on the right side of the user and select "Delete" option.

Vacancies

To add a vacancy click the button "new vacancy" on the vacancies tab. New modal with vacancy fields will be open. Some of the fields are required. Required fields are:
-Speciality
-Job type
-Description
Other fields are optional.

Vacancies actions

Click vacancy action button and select action you need:
-Watch vacancy candidates
-Add new candidate
-Delete a vacancy
-Edit a vacancy

Vacancy's candidates

Vacancy can contain some candidates, you can watch them by selection "Watch candidates" option in vacancy action button.

Creation of candidate

Creation of candidate looks alike worker profile creation but there is some differences in fields. Required fields are:
-First name
-Last name
-Gender
-Hr skype
It is strongly recommended to fulfill LinkedIn url field, to help HR get additional information about candidate.

Deleting the candidates

There is no possibility to delete group of candidates, you can delete or all candidates or one. To delete all candidates click button "Delete all" and confirm your decision in opened modal. To delete one you should open candidate actions (on the right side of candidate) and select "Delete" option.
